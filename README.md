# Features
- Bootstrap 4.3 
- Bootstrap 4.3 Validation
- Verified by W3C
- 3 Pages included (Login, Register, Forgot Password)
- No custom classes
- Easy to use
- Cross-browser compatibility (Chrome, Firefox, Opera, IE11, Safari)
- Fully responsive
- and more ...
